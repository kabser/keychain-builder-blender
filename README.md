# Keychain Builder for Blender

**by Sergey Kabanov (@kabser)**

> 🇷🇺 [Русская версия ниже](#keychain-builder-для-blender)

---

Generate 3D-printable text keychains in one click directly inside Blender.  
Type your text, pick a font, press **Build** — get a ready-to-print STL.

![Keychains with names](https://raw.githubusercontent.com/kabser/keychain-builder-blender/main/images/Пост_1_Имена.png)
![Keychains with phone numbers](https://raw.githubusercontent.com/kabser/keychain-builder-blender/main/images/Пост_-_телефоны.png)
![Keychains with game themes](https://raw.githubusercontent.com/kabser/keychain-builder-blender/main/images/Пост-Игры.png)

---

## Features

- Any text, any TTF/OTF font — including Cyrillic
- Automatic base shape that follows letter outlines
- Side ear with a round hole for the key ring
- Adjustable width, thickness, margins, bevel and more
- One-click STL export
- Works with Blender 4.0 and newer

---

## Installation

1. Download `Keychain_Builder_for_Blender_1_0.py`
2. In Blender: **Edit → Preferences → Add-ons → Install**
3. Select the downloaded file and enable the addon
4. Open the **N-panel** in the 3D Viewport → tab **Keychain**

---

## Quick Start

1. Type your text in the **Text** field
2. Click the folder icon to pick a font (TTF or OTF)
3. Set the desired width in mm
4. Press **Build Joined Filled Pads**
5. Press **Save STL** to export

---

## Requirements

- Blender 4.0 or newer
- Any TTF or OTF font file

# Keychain Builder для Blender

**Автор: Sergey Kabanov (@kabser)**

Генерируй брелоки с текстом для 3D-печати прямо в Blender, в один клик.  
Введи текст, выбери шрифт, нажми **Build** — получи готовый STL.

---

## Возможности

- Любой текст, любой шрифт TTF/OTF — включая кириллицу
- Автоматическое основание, которое повторяет контур букв
- Боковое ушко с круглым отверстием под кольцо
- Настраиваемые ширина, толщина, отступы, скругления и многое другое
- Экспорт STL одной кнопкой
- Работает с Blender 4.0 и новее

---

## Установка

1. Скачай файл `Keychain_Builder_for_Blender_1_0.py`
2. В Blender: **Edit → Preferences → Add-ons → Install**
3. Выбери скачанный файл и включи аддон
4. Открой **N-панель** в 3D Viewport → вкладка **Keychain**

---

## Быстрый старт

1. Введи текст в поле **Text**
2. Нажми на иконку папки и выбери шрифт (TTF или OTF)
3. Укажи нужную ширину в мм
4. Нажми **Build Joined Filled Pads**
5. Нажми **Save STL** для экспорта

---

## Требования

- Blender 4.0 или новее
- Любой шрифт в формате TTF или OTF
